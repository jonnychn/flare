Mini Flare

Mini Flare is a web-based lighting app that offers multiple modes for different lighting needs. It provides emergency flashing lights, mystical color transitions, and a comfortable reading light designed to minimize blue light exposure before bedtime.

Features

Emergency Mode: Flashes between red and white light at an adjustable speed, designed for alert situations. Users can control the flashing speed to adapt to different environments.

Mystical Mode: Smoothly transitions between multiple colors to create a calming, mystical atmosphere.

Reading Mode: Provides a warm light ideal for reading, especially before sleep. The reading temperature is adjustable to minimize blue light exposure and reduce eye strain.

Color Palette: A simple color palette allows users to select different colors to illuminate the screen for various uses.

Collapsible Controls: The app features a collapsible control panel, allowing users to hide controls to maximize the screen area when using the light.

Getting Started

Clone or download this repository.

Open the index.html file in a web browser.

Use the available controls to select your desired mode and customize the settings.

Usage

Emergency Mode: Click the Emergency Mode button. Use the Flashing Speed slider to adjust the rate of flashing.

Mystical Mode: Click the Mystical Mode button to initiate a relaxing color transition.

Pause: Click Pause to stop any active effects.

Reading Mode: Click the Reading Mode button and use the Reading Temperature slider to adjust the warmth of the light.

Color Palette: Select a color from the color palette to change the background color instantly.

Show/Hide Controls: Click the Show/Hide Controls button to toggle the visibility of the control panel.
